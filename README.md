# Pointer-Security
A security system for computers which runs as the background process and tracks keyboard or mouse inputs. Any unauthorized input triggers this system and sends a notification to the userâ€™s phone through text messages (GSM). Users can use a mobile app (named Pointer Security) to see the live webcam feed, screen streaming and check who is using the computer and can send commands remotely through the app to either shut down or lock the computer. The system uses socket programming where the server is developed in C#. The mobile application is developed for windows and android platform. RTP protocol is used to send the live feed.

# How to Run: 
The time server has to be running in the computer. The .xap file can be intalled in a windows phone to run this app.
